# Тестирование API сервиса «Яндекс.Прилавок» (Postman)

## Описание проекта

API-тестирование сервиса доставки продуктов. Проверены ручки: наборы (kits), быстрая доставка (fast-delivery), корзина (orders) и другие.

**Какую проблему решает проект?**  
Показывает, что я умею тестировать API на уровне запросов и ответов: позитивные и негативные сценарии, граничные значения, проверку статус-кодов и структуры JSON/XML.

**Стек (инструменты):**  
- Postman (ручное выполнение запросов)  
- JSON, XML  
- Техники тест-дизайна (граничные значения, классы эквивалентности)

## Вывод

Проект демонстрирует моё понимание API-тестирования: умение проектировать проверки, анализировать ответы и документировать дефекты.
